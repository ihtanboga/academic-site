Stent Expansion Comparator is a clinician-focused coronary stent sizing tool that helps you compare nominal versus maximum expansion capacity across drug-eluting and specialty stents side by side, using manufacturer Instructions for Use (IFU) so post-dilatation choices stay within labeled limits.

Use the Target Diameter Finder to enter a desired vessel size (mm) and instantly list stents that can safely reach or exceed that target, then browse by stent name or manufacturer, filter by nominal diameter, and add up to 8 stents for a clear bar-chart and table comparison. Built for everyday PCI planning, including large-vessel and left main sizing, it supports evidence-informed stent selection where understanding overexpansion behavior matters.
